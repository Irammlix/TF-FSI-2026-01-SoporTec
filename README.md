# TF-FSI-2026-01-SoporteTec
Sistema de Gestión de Soporte Técnico Interno desarrollado en C# (Windows Forms) con arquitectura de 3 capas, Entity Framework y SQL Server. Implementa un sistema de tickets con módulos diferenciados para solicitantes, técnicos y administradores. Trabajo Final del curso Fundamentos de Sistemas de Información - UPC 2026-10.
